# ArknightsLauncher

## 简介

>此项目仅使用C#对[[SkyHao723]](https://github.com/SkyHao723)大佬的[arknightsSwitcher](https://github.com/SkyHao723/arknightsSwitcher)进行重写,整合添加了图形化操作界面并。

## 注意事项
- 软件的选择根目录缓存在(```C:\Users\用户名\AppData\Local\ArknightsLauncher\```),名字为```config.json```，如果需要重置软件请删除此文件

## 如何使用
在Releases下载最新版本的安装包，安装完成后打开软件，选择游戏根目录，点击切换即可

## 已支持功能
- [x] 双服切换
